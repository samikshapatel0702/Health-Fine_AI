# 🩺 HealthFine AI

![HealthFine AI Logo](logo.png)

## 📌 Overview

HealthFine AI is a web-based healthcare awareness platform designed to help users understand common lifestyle diseases, assess possible health risks, and receive personalized wellness recommendations.

The project aims to promote preventive healthcare through interactive tools, educational content, and simple health analysis features.

## 🚀 Features

### 🔍 Disease Risk Calculator
- Analyze risk levels based on selected symptoms
- Covers multiple lifestyle-related diseases:
  - Asthma
  - Cancer
  - Depression
  - Diabetes
  - Heart Disease
  - Hypertension
  - Obesity
  - PCOS
  - Stroke
  - Thyroid Disorders

### 📝 Lifestyle Risk Quiz
- Quick health lifestyle assessment
- Personalized risk feedback

### ⚖️ BMI Calculator
- Calculate Body Mass Index
- Health category classification
- Basic health recommendations

### 💧 Water Intake Tracker
- Monitor daily hydration levels
- Progress tracking system

### 📚 Health Articles
- WHO-based educational resources
- Disease awareness and prevention information

### 🌙 Dark Mode
- User-friendly dark theme support

### 💬 Feedback System
- Collect user feedback and ratings

## 🛠️ Technologies Used
